# NACPA Dinner Invitation Collection

A curated collection of elegant, production-grade dinner invitation websites for the North American Concert Promoters Association (NACPA).

## Overview

This project contains multiple sophisticated invitation designs featuring:
- Modern, elegant HTML/CSS layouts
- Responsive design for all devices
- NACPA brand color palette integration
- Professional typography and spacing
- Interactive elements and hover effects

## Color Palette

The designs utilize the NACPA brand colors:
- **Teal Blue**: #345c65
- **Slate Blue**: #517283
- **Sage Taupe**: #767769
- **Soft Blue-Gray**: #aabbbf
- **Warm Beige**: #c6c6c0
- **Off-White**: #eeeeec

## Design Collections

### 1. Gala Variation Series
- `index.html` - Primary gala design (Los Angeles, April 20th)
- `nacpa-gala-variation-2.html` - Alternative design
- `nacpa-gala-variation-3.html` - Third variation

Each features a 2/3 + 1/3 split layout with event details and evening program schedule.

### 2. Executive Dinner Series
- `nacpa-exec-dinner-variation-1.html` - Sage taupe primary
- `nacpa-exec-dinner-variation-2.html` - Slate blue primary
- `nacpa-exec-dinner-variation-3.html` - Teal blue primary

Elegant centered card designs with double-border frames and detailed logistics grids.

## Features

- **Responsive Design**: Mobile-first approach with breakpoints for tablets and desktops
- **Tailwind CSS**: Utility-first CSS framework for rapid development
- **Google Fonts**: Playfair Display, Montserrat, Crimson Pro, and Manrope typography
- **Material Icons**: Professional icon library integration
- **RSVP Functions**: Interactive calendar and attendance confirmation elements
- **Venue Information**: Location details with map integration
- **Event Schedule**: Comprehensive timeline of activities

## Usage

Each HTML file is standalone and can be deployed directly. Simply open in a web browser or host on any web server.

### Customization Options:
- Update event dates and times
- Change venue information
- Modify host names and details
- Adjust RSVP deadlines
- Update contact information in footer

## Color Variants
